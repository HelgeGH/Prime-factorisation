# Prime factorisation
Gives the primal representation of every integer that you input (provided that it doesn't cause an overflow).

## Input:
Any integer **n**.

## Ouput:
The primal representation of **n**.

### Examples:

**Input:**
```
6332
2147483647
13
-672474
-2345623421
```

**Output:**
```
2^2 1583
2147483647
13
-1 2 3 11 23 443
-1 23 2029 50263
```
